## Legend of 50

The Games50 Legend of Zelda game remade with Phaser.

### Run the Game

```
yarn dev
```

### Play the Game

**Controls**

`Up/Down/Left/Right` - Move

`Space` - Attack

`Enter` - Pick Up

### Assets

- Games50 coursework
- [Pot breaking SFX](https://www.soundfishing.eu/sound/ceramic-breaks)
